\# MMO Economy Platform



Учебно-портфолио проект на современном C++ для симуляции экономики онлайн-игры.



\## Цель проекта



Подготовиться к трудоустройству на позиции:



\- Junior C++ Backend Developer

\- Junior C++ Infrastructure Developer

\- Junior Game Backend Developer



\## Текущий этап



Сейчас проект находится на этапе C++ core:



\- классы;

\- инкапсуляция;

\- конструкторы;

\- RAII;

\- разделение на `.h` и `.cpp`;

\- базовая модель `Agent + Wallet`.



\## Структура проекта



\- `lesson\_01\_basics/` — текущий Visual Studio C++ проект.

\- `docs/obsidian/` — учебные заметки и документация.
