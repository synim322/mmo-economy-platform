\## Что понял



`Agent` владеет `Wallet`, потому что кошелёк является частью агента.



Поля должны быть `private`, чтобы внешний код не мог сломать состояние объекта напрямую.



Например, баланс нельзя менять напрямую. Для этого нужны методы `Deposit`, `Withdraw`, `CanSpend`.

## Улучшение Withdraw

Сегодня я изменил Withdraw так, чтобы метод возвращал результат операции.

Раньше метод ничего не сообщал вызывающему коду. Теперь вызывающий код может понять, прошла операция или нет.

Это важно для будущего backend API, потому что сервис должен возвращать понятную ошибку, если транзакция отклонена.
